# MyNavigation

Este projeto foi desenvolvido no início dos meus estudos em React Native. Ele serve como um exemplo prático de navegação entre telas e organização de componentes em um app mobile.

## O que tem neste projeto?

- Estrutura de navegação entre páginas
- Componentes organizados em pastas
- Telas de Contatos, Informações e AppContacts

## Estrutura de Pastas

- `src/Pages/AppContacts/`
- `src/Pages/Contacts/`
- `src/Pages/Information/`

## Como executar o projeto

1. **Clone o repositório:**
   ```bash
   git clone <url-do-repositorio>
   ```
2. **Instale as dependências:**
   ```bash
   npm install
   ```
3. **Execute o projeto:**
   ```bash
   npx react-native run-android # Para Android
   npx react-native run-ios     # Para iOS (MacOS)
   ```

Certifique-se de ter o ambiente React Native configurado na sua máquina. Para mais detalhes, consulte a [documentação oficial](https://reactnative.dev/docs/environment-setup).
